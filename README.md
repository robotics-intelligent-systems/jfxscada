# Supervisory Control and Data Acquisition

OpenMUC is a software framework based on Java and OSGi that simplifies the development of customized monitoring, logging and controlling systems. jMBus is an implementation of the M-Bus wired and wireless protocols. jEEBus.SHIP implements the SHIP specification version 1.0.1. jEEBus.SPINE implements Smart Premises Interoperable Neutral-Message Exchange (SPINE) in Java. Open source EEBUS (SHIP and SPINE) implementation written in C. libiec61850 is an open-source (GPLv3) implementation of an IEC 61850 client and server library implementing the protocols MMS, GOOSE and SV. A Java IEC 61850 MMS stack for clients and servers. j60870 is a library implementing the IEC 60870-5-104 communication standard. jSCADA served multiple times as a starting point for research projects. OpenSCADA is an open implementation of SCADA (Supervisory Control And Data Acquisition) and HMI (Human-Machine Interface) systems. Archimedes is an open-source Python framework designed for deployment of control systems to hardware. Open-source warehouse/manufacturing development framework and simulator. VOLTTRON Distributed Control System Platform. OpenEMS is a modular platform for energy management applications. Rapid SCADA is full featured SCADA software that supports MODBUS and OPC. Tango Controls is an open source device-oriented controls toolkit for controlling any kind of hardware or software and building SCADA systems. TANGO kernel Java implementation. Open Process Library - The MTP-compatible cross-vendor Industrial Process Automation library. PyScada is a open source scada system that uses the Django framework as backend. Scada-LTS is an Open Source, web-based, multi-platform solution for building your own SCADA (Supervisory Control and Data Acquisiton) system. SAMPO is an open-source framework for adaptive manufacturing processes scheduling. SCADA data pre-processing library for prognostics, health management and fault detection of wind turbines
