# Supervisory Control and Data Acquisition

Archimedes is an open-source Python framework designed for deployment of control systems to hardware. To make this possible, it provides a comprehensive toolkit for modeling, simulation, optimization and C code generation. The Open Research Vessel Data Acquisition System (OpenRVDAS) is a software framework used for building custom data acquisition systems (DAS). Open-source warehouse/manufacturing development framework and simulator. VOLTTRON Distributed Control System Platform. OpenEMS is a modular platform for energy management applications. The Real-Time Robot Arm Simulator is an open-source project creating a KUKA KR 700 PA-inspired virtual robotic arm using Blender and Python/Rust. Multimaterial 3D Printing with Conventional Powder Metallurgy and Selective Laser Sintering. Simulating Composites Manufacturing Processes. Rapid SCADA is full featured SCADA software that supports MODBUS and OPC. Tango Controls is an open source device-oriented controls toolkit for controlling any kind of hardware or software and building SCADA systems. Open Process Library - The MTP-compatible cross-vendor Industrial Process Automation library. PyScada is a open source scada system that uses the Django framework as backend. Scada-LTS is an Open Source, web-based, multi-platform solution for building your own SCADA (Supervisory Control and Data Acquisiton) system. Apache Ditto is a software framework for digital twins, a virtual representation of devices in the real world. OPEN100 is a project that aims to provide open-source blueprints for the construction of nuclear power plants. A Modelica library for the realistic modelling of industrial control system. BlueSeer is an open source ERP / EDI project created for the general business community
